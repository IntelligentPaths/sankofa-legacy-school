# Image Drop Zone

Drop raw images into `public/raw/` — they'll be processed into the correct
folders under `public/images/` by Claude Code on request.

## Naming convention after processing

- `public/images/students/identity-african.jpg` — student imagery for Identity pillar (African heritage)
- `public/images/students/identity-latino.jpg` — student imagery for Identity pillar (Latino heritage)
- `public/images/students/creation-robotics.jpg` — students doing project-based work (Creation pillar)
- `public/images/founders/juleon.jpg` — founder portrait (placeholder until real photo)
- `public/images/campus/exterior.jpg` — building exterior (placeholder)
- `public/images/og/default.jpg` — 1200x630px social sharing image
